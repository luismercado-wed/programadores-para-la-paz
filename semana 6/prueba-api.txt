Descripción de la respuesta devuelta por el servidor:

Al enviar una petición POST a la ruta 'http://localhost:3000/registro' con el JSON de prueba, el servidor procesa los datos del cuerpo de la petición gracias al middleware express.json() y responde con un código de estado 200 OK devolviendo el siguiente objeto JSON:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto confirma que el servidor interceptó los datos enviados de manera exitosa mediante 'req.body', los mapeó correctamente a las variables locales y generó una respuesta estructurada idéntica a la información original de entrada.
